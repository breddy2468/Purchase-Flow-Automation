Purchase Flow Automation – Project Repository

This repository contains the Selenium automation framework created to automate
the purchase flow on https://www.mystoredemo.io/.

The framework is developed using:

• Selenium WebDriver
• Java
• Page Object Model (POM) Design Pattern
• WebDriverWait for synchronization

The automation covers the following scenarios:

1. Happy Path – Complete purchase flow from product selection to order confirmation.
2. Cart Validation – Verify the selected product details match the cart.
3. Payment Validation – Validate credit card payment and success message.
